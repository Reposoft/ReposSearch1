ReposSearch    

This Open Source project delivers Fulltext search to Subversion repositories. Document formats like PDF and Word are supported, thanks to Apache Solr and Tika.


Screenshot of Repos Search integrated into Repos Style

Getting started

Download, then read the documentation.

Components

Repos Search consists of the following components, all fully customizable:

  • A simple UI that is easy to integrate into HTML pages
  • An indexing script to be executed from Subversion post-commit hooks
  • A server running Solr with a schema for fulltext and Subversion properties
